# CI-CD-with-Azure
Azure app services integrated with CI/CD techniques

Technologies: Microsoft Azure, Azure App Service, GitHub Actions, Node.js, CI/CD, Git, Linux

This project focused on deploying and managing cloud-hosted web applications using Microsoft Azure App Service and GitHub-based CI/CD workflows. The environment involved configuring automated deployment pipelines through GitHub Actions, managing Node.js application deployments, and troubleshooting build and deployment workflow issues within cloud-hosted environments. Azure App Service was used to host and manage application infrastructure while GitHub repositories and workflow automation were integrated to support continuous deployment and version-controlled application updates. The project provided hands-on experience with cloud deployment workflows, CI/CD concepts, application hosting, and DevOps-oriented deployment automation practices.
